# Grunt Project
This is a simple Project using Grunt. It shows different Cities and Hotels.

## Install
    npm install
## Start Grunt
    grunt
    grunt serve    
    
## Plugins
Different Grunt Plugins are used
### Concat
    grunt concat
### Uglify
    grunt uglify
### CSSMin
    grunt cssmin
### Less
    grunt less
### Watch
    grunt watch
### Browsersync
    grunt browsersync
### Zip
    grunt zip
Creates a Zip File from different files
    
### JsBeautify
    grunt jsbeautifier
Beautifies JS
### HTMLLint
    grunt htmllint
Validates HTML
